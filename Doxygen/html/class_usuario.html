◆ senhaValida()

int Usuario::senhaValida ( string  password)
protected

Método para checar se a senha possui uma letra maiúscula minúscula e numero. Algoritmo:

  1. Três valores booleanos setados em false;
  2. Loop pela string; 2.1 Se um valor maiusculo é encontrado a variavel maiuscula é setada para true; 2.2 e 2.3 Análago ao 2.1 com suas respectivas especificações;
  3. Se as três variáveis ao final da iteração forem verdadeiras, a senha é válida.
